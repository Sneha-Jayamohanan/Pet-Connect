# PetConnect — Digital Pet Adoption Platform

![PetConnect Banner](https://images.unsplash.com/photo-1587300003388-59208cc962cb?w=1200&q=80&fit=crop)

## Overview

PetConnect is a multi-page pet adoption web application developed using HTML, CSS, and JavaScript. The platform enables users to browse pets, explore detailed profiles, and submit adoption applications through a responsive and interactive interface.

The application focuses on simplicity, accessibility, and smooth user interaction while demonstrating frontend development concepts such as dynamic rendering, authentication handling, filtering, form validation, and responsive UI design.

The project was developed as a second-year Computer Science Engineering academic project.

---

## Key Features

- User registration and login using browser localStorage
- Protected pages with authentication checks
- Pet browsing with live search and category filtering
- Interactive 3D pet cards with hover effects
- Adoption application form with validation and auto-filled user details
- Scroll-based animations using IntersectionObserver
- Fully responsive design for desktop and mobile devices
- Clean and user-friendly interface for easier navigation
- Dynamic pet rendering using JavaScript
- Lightweight application without external frameworks or libraries

---

## Pet Categories

The platform includes over 100 pets across multiple categories:

- Dogs
- Cats
- Birds
- Fish
- Rabbits
- Hamsters
- Turtles
- Guinea Pigs
- Horses
- Exotic Pets

Each pet profile contains details such as name, age, breed, gender, location, and description.

---

## Core Functionalities

- Browse pets by category
- Search pets instantly using live filtering
- View detailed pet information through interactive cards
- Register and log in with session handling
- Submit adoption applications with validation
- Maintain user sessions using browser storage
